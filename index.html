Get started to know,my journey to design the layout and develop this website

Quick Summary

As a part of Web Course Project at alfatraining,Deutschland, I designed and developed this website simply using HTML and CSS.Project timeline was 2 weeks. I took this as opportunity to implement my course learnings into a working model.Course instructor(Holger Steinbar)walked us through the Bootrsap framework during the course.I was fascinated by the extensive functionality of the Bootstrap and how it has made web developer's life easy. To learn the deep insights of Bootstrap, I decided to develop a Website similar to Bootstrap rather than using Bootrap's templates and in build CSS. This way I could analyse the framework deeply, enhance my knowledge as well as develop my Project.

Plan the Project

The first thing to achieve was the similar layout as Bootstrap 5 using minimal Javascript. My goal was to create a website where I can present and demo my learnings and challenges I faced during my Project deveopment, that could in any way help other people in same boat as me. Thanks to the course instructor who guided the team to focus on the planning and documentation as this step helps to organize and give a structure to our thoughts.

Initial Research

To get the juice flowing, I did an initial research. I followed reverse engineering approach. I started by understanding how Bootstrap Website is developed and read some articles on research engineering to gather the required knowledge, how to analyse the website by looking at the source code of the website. I took some printouts,a pen and brainstormed my own ideas and thoughts.

Steps to follow

  • My goal was to design and develop the layout to structure the content in elements header,navigations,footer,sliders using HTML and provide styles as well as formatting using CSS.
  • My biggest tool in this whole journey is dev toolbar and Dreamweaver. At every step I unit tested the template to find the bugs at initial stage of development
  • Open the Bootstrap v5 website
  • Open the Inspector tab in dev toolbar. This tab can be used to run and edit the CSS on the fly, making changes to the CSS snippets across the page.
  • I analyzed that container is the fundamental building block of the Bootsrap and the most juicy part. My own CSS contains 2 containers
    1. container, which sets a max-width at each responsive breakpoint
    2. container-fluid, which is width: 100% at all breakpoints
  • Main components of the Bootsrap5 website in body tag are,
    1. header
    2. subnavbar(subnavigation header)
    3. Main Content container that consists Sidebar + main content
    4. main class consists of introduction, definition,Table of contents
  • Bootstrap 5 is implemented using Flex box and Grid.

How my approach is different from Bootstrap5 Website

  • Bootstrap5 website is developed to be mobile-first and uses media queries to create breakpoints for layouts. Breakpoints are based on minimum viewport widths and allow to scale up the elements as needed. See breakpoints to get deep insights. Although Mobile first is a modern approach but I followed Desktop first approach to have hands on. Mobile first approach, will implement in upcoming Projects.

Challenges

  • Determine the layout : Bootstrap website layout is a mix of Flex and Grid approach. To set the elements under each other(1 dimensional) flex-box approach is used and to set the elements alternate to each other(2-dimensional) Grid approach is used.
  • If you scroll down,you will notice that 2 side Navigation bars left and right are sticky. Although to make an element sticky is not that difficult but to make flex elements sticky was quite tricky.
  • Grid Layout is very extensive and need additional time to demonstrate the detailed examples. Unfortunaltely,could not implement this feature in 1st phase of the project.
