What is Flex

A CSS layout that is best suited for one dimensional layout.

Why Flex Box?

  • No FLOATS!
  • Responsive and mobile-friendly
  • Positioning child elements is much easier
  • Order of elements can be changed without editing the HTML source code
  • Better way to build one-dimensional layouts
